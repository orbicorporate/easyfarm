# 🌱 Easyfarm — Planejamento Agrícola Inteligente

App PWA de planejamento agrícola para produtores brasileiros.

## 🔗 Links
- **App ao vivo:** https://easyfarm-nume-company.vercel.app
- **Banco de dados:** Supabase (sa-east-1 — São Paulo)

## ✅ Funcionalidades
- Login e cadastro com e-mail e senha
- Calculadora de lucro por cultura e variedade
- Guia de dicas por cultura
- Plano de produção com checklist
- Histórico de simulações salvo na nuvem
- PWA — instala no celular como app

## 🗺️ 18 culturas em 5 regiões do Brasil
Sul, Sudeste, Centro-Oeste, Nordeste e Norte

## 🛠️ Stack
- Frontend: React (HTML único, zero build)
- Backend: Supabase (PostgreSQL + Auth)
- Deploy: Vercel
- PWA: manifest.json + Service Worker

## 📁 Arquivos
- `index.html` — app completo
- `manifest.json` — configuração PWA
- `sw.js` — service worker (cache offline)
- `database/schema.sql` — estrutura do banco

Deploy conectado ao GitHub em 20/07/2026.
